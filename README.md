# 🐾 Pet Adoption Website

A simple and interactive **Pet Adoption Website** built using **HTML, CSS, and JavaScript**.
This project allows users to adopt pets, donate, and volunteer to help animals in need.

---

## 📌 Features

* 🐶 View available pets (Dogs & Cats)
* ❤️ Adopt a pet using a popup form
* 💰 Donate money or items
* 🙋 Volunteer signup form
* 📞 Contact form
* 📱 Responsive design
